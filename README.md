# uw-test

This repository contains both the **client** (React) and **server** (Node.js) parts of the project in one place.

## Setup Instructions

### Installation

#### Client
```bash
cd client
yarn install
```
#### Server
```bash
cd server
yarn install
```
### Run Project

#### Client
```bash
cd client
yarn start
```

#### Server
```bash
cd server
yarn start
```

### Run unit tests

```bash
yarn run test
```
